# Discuss

A learning project built with Next.js for creating and discussing topics.

This app is based on a discussion forum style project originally inspired by Stephen Grider's course material. It is being used to learn Next.js features, TypeScript, Prisma, authentication, and app routing.

## Features

- Topic creation and discussion flows
- Post creation within topics
- Authentication with NextAuth / Prisma
- Nested app routes for topics and posts
- Tailwind CSS styling
- Prisma database schema and migrations

## Tech stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Prisma
- NextAuth
- Zod
- NextUI

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Notes

This repository is a learning project, not a production application. It is intended to help understand how to build Next.js apps with authentication, database integration, and nested routes.
